# CV Web Paula Liñán Jurado

La página web empieza con el CV en inglés por defecto, desde arriba a la derecha (donde las banderas) se puede 
cambiar el idioma entre catalán, castellano o inglés.

En el mismo header también tenemos el botón del modo día/noche, cuando estamos en el modo día es una luna y cuando 
estamos en el modo noche un sol.

Después tenemos el menú, el qual se mantiene fijo en la misma posición aunque bajemos la pantalla y enlaza cada 
parte de la web. Este menú se vuelve un menú desplegable cuando pasamos a formato tableta o móvil.

A lo largo de toda la web tenemos una divisón entre el main y el aside. En el aside tenemos la información de 
contacto junto con mi foto, los enlaces a mis redes sociales y un botón para descargar el CV en formato PDF, debajo 
está lo que he titulado "sobre mí" que es un párrafo donde explico mis Soft Skills, si se pasa por encima de cada 
dibujo dice a que Soft Skill hace referencia. Debajo de esto están los idiomas. Tanto el apartado "sobre mí" como 
el apartado "idiomas" hay que pasar por encima con el ratón para que aparezca el texto.

En el main empezamos con los estdios realizados, después la experiencia laboral, después las Hard Skills, donde si 
pasamos por encima de cada dibujo dice qué Hard Skill es. Por último tenemos un carroussel con los proyectos más 
destacados que he realizado, si pasamos por encima de cada uno y clicamos nos conduce a su página de Github excepto 
el primero que conduce a la página donde está subido para que se pueda jugar.

Una vez terminado el main y el aside nos queda el footer, donde tenemos el formulario de contacto, validado con 
Regex, debajo la fecha y hora a la que se ha accedido a la web y debajo de esto el copyright. A la derecha del 
footer (debajo de todo esto si estás en el modo móvil) está el easter egg, el cual es un huevo que si clicas 3 
veces se rompe y sale un avestruz enfadado.
